# 图片像素格转换器

一个使用 React + Vite + JavaScript 开发的纯前端图片像素格转换器。用户可以上传本地图片，在浏览器内通过 HTML Canvas 生成像素格效果，并导出 PNG 图片。

## 安全说明

所有图片处理都在浏览器本地完成，不会上传用户图片。本项目不包含后端、数据库、登录系统、云存储或支付功能。

## 功能

- 上传一张本地图片并预览原图。
- 实时查看像素格处理结果。
- 调整像素块大小：4、8、12、16、24、32。
- 选择颜色数量：原图颜色、8 色、4 色、2 色。
- 显示或隐藏网格线。
- 调整网格线颜色和粗细。
- 显示横向格数、纵向格数和总格数。
- 将处理后的图片导出为 PNG。
- 手机端单列布局，适合手机浏览器使用。
- 支持基础 PWA，可部署后添加到手机主屏幕。

## 技术栈

- React
- Vite
- JavaScript
- HTML Canvas
- Web App Manifest
- Service Worker
