# 图片像素格转换器

一个使用 React + Vite + JavaScript 开发的纯前端图片像素格转换器。用户可以上传本地图片，在浏览器内通过 HTML Canvas 生成像素格效果，并导出 PNG 图片。

## 功能

- 上传一张本地图片并预览原图。
- 在右侧实时查看像素格处理结果。
- 调整像素块大小：4、8、12、16、24、32。
- 自动计算并展示横向格数、纵向格数和总格数。
- 选择颜色数量：原图颜色、8 色、4 色、2 色，让图片更接近拼豆、十字绣或像素图纸效果。
- 显示或隐藏网格线。
- 调整网格线颜色和粗细。
- 将处理后的图片导出为 PNG，导出图包含当前像素格统计标注。
- 所有图片处理都在浏览器本地完成，不会上传用户图片。

## 技术栈

- React
- Vite
- JavaScript
- HTML Canvas
- 本地颜色量化算法

## 本地安装

请先确认本机已安装 Node.js 和 npm。

```bash
npm install
```

## 本地运行

```bash
npm run dev
```

启动后，根据终端提示打开本地地址，通常是：

```text
http://localhost:5173/
```

## 构建生产版本

```bash
npm run build
```

构建产物会输出到 `dist/` 目录。

## 预览生产构建

```bash
npm run preview
```

## 本地测试新功能

1. 运行 `npm install` 安装依赖。
2. 运行 `npm run dev` 打开页面。
3. 上传任意本地图片，观察右侧预览上方的当前设置摘要，确认会显示像素块大小、颜色数量、横向格数、纵向格数和总格数。
4. 切换像素块大小，确认横向格数按 `Math.ceil(imageWidth / pixelSize)` 更新，纵向格数按 `Math.ceil(imageHeight / pixelSize)` 更新。
5. 切换颜色数量为 8 色、4 色、2 色，确认像素图颜色会减少；切回“原图颜色”则只保留像素块处理。
6. 调整网格线显示、颜色和粗细，确认预览自动更新。
7. 点击“导出 PNG 图片”，确认导出图片与当前预览一致，并在图片上方留白区域包含横向、纵向和总格数标注。
